A better way to search for anything on the WordPress dashboard.

== Description ==

WP Spotlight Search plugin allows you to search anything on your WordPress admin quickly, precisely and efficiently. No hassle! Nothing!

= Everything in One Search =
You can search everything like menus, submenus, posts, pages, comments, users, products, orders and etc,

== Installation ==

1. Upload the plugin folder to your /wp-content/plugins/ folder
2. Go to the Plugins page and activate WP Spotlight Search
